Easily create and manage FAQs on your WordPress site. Display FAQs with a shortcode or on dedicated pages.

== Description ==

FAQ Info Instigator provides a simple and efficient way to create and manage FAQs on your WordPress site. You can display FAQs on any page or post using shortcodes. Choose to show all FAQs at once, filter them by category, or list available categories for easy navigation.

**Try it out on your free dummy site: Click here => [https://tastewp.org/plugins/faq-info-instigator](https://tastewp.org/plugins/faq-info-instigator)**

== Features ==

FAQ Info Instigator offers a range of powerful features to help you create, manage, and display FAQs on your WordPress site:

* **Custom Post Type**: Easily create and manage FAQs with a dedicated custom post type.
* **Shortcodes**: Display FAQs anywhere on your site using simple shortcodes.
  * `[faq_info_instigator]`: Display all FAQs.
  * `[faq_info_instigator category="your-category-slug"]`: Display FAQs from a specific category.
  * `[faq_info_instigator_categories]`: Display a list of all FAQ categories.
* **Category Filtering**: Organize FAQs into categories for better navigation and display.
* **Flexible Display Options**: Choose to show all FAQs, filter them by category, or list available categories for easy user access.
* **Easy to Use**: Intuitive interface for adding and managing FAQs.
* **Responsive Design**: FAQs are displayed beautifully on all devices.
* **Compatibility**: Tested up to the latest version of WordPress (6.5.4) and requires PHP 5.6 or later.

These features make FAQ Info Instigator an ideal choice for any website looking to provide a comprehensive FAQ section for their users.

== Installation ==

1. Upload the `faq-info-instigator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the provided shortcodes in your posts or pages to display FAQs.

== Frequently Asked Questions ==

= How do I display all FAQs? =

Use the `[faq_info_instigator]` shortcode to display all FAQs.

= How do I display FAQs by category? =

Use the `[faq_info_instigator category="your-category-slug"]` shortcode to display FAQs from a specific category. Replace `your-category-slug` with the actual slug of the category you want to display.

= How do I display a list of FAQ categories? =

Use the `[faq_info_instigator_categories]` shortcode to display a list of FAQ categories.
